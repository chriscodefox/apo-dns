# apo-dns

Apotheca RDNS daemon.

    Usage: apo-dns [-h|--help] [-p|--port PORT] [-s|--ssl PORT]
        Run the Apotheca Ring Domain Name System Daemon
        
Run as a [daemon][1]

Visit [https://apo-dns.com] for more information.
